This is an AppImage of the Linux Creative Suite AppPack installer. It offers alternatives to the Adobe Creative Cloud suite of software to be selected and installd with a GUI and provides custom icon and custom program grouping in the Linux DE application menu.

The following applications are available to be installed by the distribution's native package manager or flatpack as a fallback.

* GIMP
* Inkscape
* Kdenlive
* Blender
* Krita
* LibreOffice Draw
* Scribus
* OpenToonz
* Synfig Studio
* Audacity
* darktable
* RawTherapee
* BlueGriffon
* Figma
* Penpot
* Okular
